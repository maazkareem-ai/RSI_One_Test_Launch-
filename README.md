# RSI ONE - Aviation Asset Management System Backend

Complete backend API for managing aviation assets, flight logs, maintenance, expenses, and document processing with AI/OCR capabilities.

## 🚀 Features

### Authentication & Authorization
- JWT-based authentication via Supabase
- Role-based access control (Owner, Manager, Pilot, RSI Admin, Technician)
- Login/Logout, Forgot Password, Password Reset
- User activity logging
- IP-based rate limiting

### Asset Management
- CRUD operations for assets (Jets, Turboprops, Helicopters, Yachts)
- Asset metadata tracking (Model, Serial Number, Registration, etc.)
- File attachments (manuals, certificates, images, documents)
- Automatic expiry reminders (Insurance, ARC, CofA)

### Flight Log Engine
- Create, edit, delete flight logs
- CSV bulk upload
- Automatic calculations for Flight Hours (FH) and Cycles
- Asset utilization statistics and charts
- Pilot tracking

### Maintenance Log System
- Track all maintenance types (A-check, C-check, SB, AD, Unscheduled)
- Component replacement tracking
- Technician details and licensing
- AI predictions for next maintenance
- Overdue and upcoming maintenance alerts

### Expense Management
- Track all operational expenses
- Multi-currency support
- Invoice management with attachments
- Cost categorization
- AI anomaly detection
- Duplicate invoice detection

### OCR & Document Processing
- Upload PDF, PNG, JPG, HEIC, Word, Excel
- AI-powered data extraction
- Manager approval workflow
- Confidence scoring
- Error flagging for review
